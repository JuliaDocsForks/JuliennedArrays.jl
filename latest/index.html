JuliennedArrays.combine — Method.
combine(array)

Combine an array that has been julienned (or otherwise split).

julia> using JuliennedArrays

julia> array = reshape(1:9, 3, 3)
3×3 Base.ReshapedArray{Int64,2,UnitRange{Int64},Tuple{}}:
 1  4  7
 2  5  8
 3  6  9

julia> begin
            julienned = julienne(array, (:, *))
            mapped = map(sum, julienned)
            combine(mapped)
        end
1×3 Array{Int64,2}:
 6  15  24

JuliennedArrays.julienne_iterator — Method.
julienne_iterator(array, julienne_code)

julienne_code should be a tuple of either * (for dimensions to be sliced over) or : for dimenisons to be sliced across. See the example below for clarification.

julia> using JuliennedArrays

julia> array = reshape(1:9, 3, 3)
3×3 Base.ReshapedArray{Int64,2,UnitRange{Int64},Tuple{}}:
 1  4  7
 2  5  8
 3  6  9

julia> begin
            iterator = julienne_iterator(array, (:, *))
            map(identity, iterator)
        end
3-element Array{Tuple{Base.OneTo{Int64},Int64},1}:
 (Base.OneTo(3), 1)
 (Base.OneTo(3), 2)
 (Base.OneTo(3), 3)
